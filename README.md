# 学术论文爬虫系统 (Java版)

一个基于Java的高性能学术预印本爬虫系统，支持多线程、异步和分布式爬取模式。

## 功能特性

- 🔥 **多种并发模式**: 支持线程、异步、分布式三种并发模型
- 📚 **多数据源支持**: 同时爬取arXiv、bioRxiv和ChemRxiv论文数据
- ⚙️ **灵活配置**: 基于TOML的配置文件，支持动态调整爬取参数
- 🏗️ **现代Java特性**: 使用Java 11+特性，模块化设计，低耦合度架构
- 📊 **多种存储格式**: 支持JSON文件存储和数据库存储
- 🎯 **智能分类**: 按物理、材料、化学、生物学等学科关键词自动分类

## 快速开始

### 环境要求

- Java 11 或更高版本
- Maven 3.6 或更高版本
- 网络连接（用于访问论文API）
